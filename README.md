# ASOCIACIÓN ALUMA
Se solicita una aplicación que facilite la gestión las excursiones de una asociación (ALUMA).  


La aplicación debe ser capaz de:  


-  Crear excursiones con:  

    -  Denominación de la excursión  

    -  Ciudad de Destino
    -  Actividad a desarrollar
    -  Tipo de Trasporte
    -  Personas apuntadas 
    -  Numero de plazas iniciales y plazas disponibles
    -  Fecha
    -  Manutención 
    -  Si hubiera pernocta:  

        -  Días de duración
        -  Tipo de alojamiento.  

•	Según el tipo de transporte se podrá reservar asiento (ejemplo autobus).  

•	Según tipo de persona apuntada (socio o no) se le cobrará un precio diferente.  

•	Deberá tener capacidad de generar documentos de reserva para los apuntados ( a modo de justificante) con los datos de la excursión y de la reserva.  

•	Permitirá recuperar los datos históricos de la misma mostrando todos los datos.  

•	Debe permitir que una vez finalizada la excursión los socios puedan hacer comentarios sobre la misma   

La aplicación deberá tener un sistema de suscripción de alertas en el que los socios podrán apuntarse para recibir las novedades de la asociación.

La aplicación será accesible desde cualquier dispositivo
